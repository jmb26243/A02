How to create an Account and Repository on GitHub

1. Search for GitHub on browser
2. Click the link
3. Select "Sign Up"
4. Choose login method (e.g., Google, Apple, manual, etc.)
5. Enter login information
6. Complete security puzzle
7. After completing 1-6 properly, user should be directed to the homepage of GitHub
8. Navigate to the left side of the screen and click "Create Repository"
9. Adjust reposiory settings if needed
10. Create the repository
